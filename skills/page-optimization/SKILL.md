# Page Optimization Skill

Use this skill when improving any Forever Forward website page.

## Objectives

- Keep the tone bold, succinct, aggressive, and minimal.
- Preserve the brand guide colors, typography, and disciplined visual tone.
- Keep one primary CTA per page.
- Make every page ready for Calendly, Stripe, or form integration where applicable.

## Workflow

1. Read the target page and its matching file in `agents/pages/`.
2. Check for conflicts with the main conversion goal: booking a free consultation.
3. Confirm the page still matches the roadmap and brand guide PDFs.
4. If the page has a mistake or weak spot, add a short note to its `Learning Log`.
5. Escalate to `agents/executive/director_brand.md` or `agents/executive/director_conversion.md` when the issue crosses page boundaries.

## Success Criteria

- The page message is instantly clear.
- The page has one dominant action.
- Copy sounds strong without sounding inflated.
- The page fits the site visually and structurally.
