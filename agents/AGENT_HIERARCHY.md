# Forever Forward Agent Hierarchy

This project uses a page-expert model. Every page has a dedicated agent that owns clarity, conversion, and consistency for that page. Page agents report to directors. Directors report to the chief orchestrator. The chief orchestrator answers to the project lead.

## Chain of Command

1. Project Lead
   Codex or the active human decision-maker sets priorities, approves tradeoffs, and resolves conflicts.
2. Chief Orchestrator
   Maintains the full site system, coordinates directors, and enforces launch priorities.
3. Directors
   Brand Director and Conversion Director review cross-page alignment.
4. Page Agents
   Home, About, Training, Military Prep, Speaking, and Consultation agents own page-level expertise.

## Communication Rules

- Page agents must notify both directors when a change affects brand voice and conversion flow.
- Directors reconcile conflicts before escalating upward.
- Any repeated issue becomes a documented learning in the relevant page agent file.
- When a page agent makes a mistake, it must log the failure, root cause, fix, and future guardrail.

## Learning Loop

1. Detect mistake or inconsistency.
2. Record it under `Learning Log` in the page agent file.
3. Share the pattern with the relevant director.
4. Update any workflow skill if the lesson applies across multiple pages.
